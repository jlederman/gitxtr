gitxtr

Cross-platform commit graph viewer

gitxtr is a native commit graph viewer built for developers who need to see their Git history clearly. Canvas-rendered branch graph, syntax-highlighted diffs, file history, blame, interactive rebase, and 527 color themes — no Electron, no browser tab.

Visual commit graph

Canvas-rendered branch lanes with merge lines, ref chips, and keyboard navigation through thousands of commits without slowing down.

Unified & split diff

Syntax-highlighted diffs in both unified and side-by-side modes, with full file history and line-level blame.

527 color themes

Every iTerm2 color scheme, live-previewed in the settings panel. Arrow-key selectable without reopening the dropdown.

Cross-platform, native

Runs on macOS, Windows, and Linux via .NET and WebKit. Fast startup, small footprint — no runtime to install.
